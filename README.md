# outlook-afterhours
Prompt if emails sent after hours should be delayed\
Working hours are M-F before 7am or after 6pm, local time.

## Step 1:
Download .zip and extract to a folder

## Step 2:
Close Outlook\
Execute AfterHoursInstall.bat
